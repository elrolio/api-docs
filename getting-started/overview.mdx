Everything you do in Descript — removing filler words, enhancing audio, adding captions, creating highlight clips — is now available through code. Or through an AI assistant. Or through a Zapier zap. The editing intelligence stays the same. The interface adapts to you.

## Three ways in

You don't have to choose between power and convenience. The API meets you where you work.

## What you can do

**Import media** — Create a new project and import media from URLs in a single request. Descript handles transcription, processing, and composition setup automatically.

**Edit with AI** — Use natural language prompts to edit any project. The Agent API accepts the same kinds of instructions you'd give Underlord in the app: remove filler words, apply Studio Sound, add captions, create highlight clips, translate videos, add B-roll, or generate new content entirely from a text prompt.

**Check job status** — Both import and editing operations run asynchronously. Poll for status or provide a callback URL to get notified when a job completes.

<Info>
There is no export endpoint yet. After editing via the API, open the project in Descript to export your final output.
</Info>

Ready to try it? The [Quickstart](/getting-started/quickstart) walks you through all three operations with code samples in cURL, Python, and Node.js.

## How it works

Everything is asynchronous. You submit a request — import media, run an agent edit — and get back a `job_id`. Then you either poll for the result or hand Descript a callback URL and go do something else.

```
Request → Job created → Processing → Completed → Open in Descript
```

Jobs are retained for 30 days. See [Working with Jobs](/guides/working-with-jobs) for polling patterns, job states, and webhook callbacks.

## Authentication

API requests use bearer token authentication. Create a token in your Descript account settings — each token is scoped to a single Drive.

Using an AI assistant? The MCP connector handles auth via OAuth. No token needed.

See [Authentication](/getting-started/authentication) for setup instructions.
